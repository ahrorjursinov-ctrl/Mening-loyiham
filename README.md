 <p>BMW (Bayerische Motoren Werke) — Germaniyaning mashhur avtomobil ishlab chiqaruvchi kompaniyasi. U 1916-yilda tashkil etilgan. Bosh qarorgohi Myunxen shahrida joylashgan.
